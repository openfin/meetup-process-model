Create a new Window

const win = new fin.desktop.Window({
        name:`cwin_${Math.floor(Math.random() * 100)}`,
        url: 'child.html',
        autoShow: true,
        saveWindowState: false
    });

Create a new Application


const uuid = `app_${Math.floor(Math.random() * 100)}`;
const app = new fin.desktop.Application({
    name: uuid,
    uuid: uuid,
    url: location.href,
    autoShow: true,
    saveWindowState: false
}, () => {
    app.run();
});

Service Worker

This demo makes use of a service worker in order to show how to handle offline functionality in OpenFin.

After the application has loaded, go to the console and kill the web server with Ctrl+C, then afterwards you can continue to create a new window and new application.
